### About

I build clean, responsive web interfaces and keep improving how I structure frontend code. Right now I work with TypeScript in low-code products and grow deeper in React, component architecture, accessibility, and reliable UI states.

- Working on UI-heavy flows with TypeScript.
- Learning React patterns through real, shipped code.
- Interested in developer tools, low-code platforms, and products that make complex work feel simple.
- Improving every week: code quality, communication, and engineering discipline.
